1. Project Introduction

No more paper registers. Smart Attendance for Smart Colleges.

Why we built this?

The Struggle: Paper sheets get lost. Friends answer for absent friends (Proxy). Teachers waste hours just counting.

The Impact: Students have NO idea they are "Short of Attendance" until they get detained. Too late.

How it works?

üë®‚Äçüè´ Teachers: Open app -> Select Class -> Mark Absent -> Done. Takes 10 seconds.

üë®‚Äçüéì Students: Check app -> See "75% Attendance" instantly. No confusion.

Is it safe?

100% Secure. Students can ONLY see their own data (Privacy).

Auto-Lock System: Once attendance is marked, it "locks" automatically after 3 days. No manual tampering possible.

4. Technical Details

RLS Security Policy

CREATE POLICY faculty_update_attendance ON attendance
FOR UPDATE USING (
  faculty_id = auth.uid() AND
  is_locked = FALSE AND
  marked_at >= NOW() - INTERVAL '3 days'
);
